Tic-Tac-Toe Game (Python + Tkinter)

This project is a graphical Tic-Tac-Toe game developed using Python and Tkinter, featuring a user-friendly interface and basic game logic. It allows two players to play alternately on the same device with real-time updates and win detection.

The game also integrates internationalization (i18n) using gettext, enabling support for multiple languages.

Features
Interactive GUI-based Tic-Tac-Toe board (3x3 grid)
Two-player gameplay (Player X vs Player O)
Automatic win detection system
Game reset functionality with a “New Game” button
Popup notification for game results
Built-in multi-language support using gettext

How It Works
The game window displays a 3×3 grid of buttons.
Players take turns clicking on empty cells:
Player 1 → X
Player 2 → O
After each move:
The game checks for a winning combination
If a player wins, a popup appears
The board resets automatically

Controls
Mouse Click – Place your mark (X or O)
New Game Button – Reset the board manually
Close Window – Exit the game
